Dataset Title
Forest Fire Prediction Dataset for Nallamala Forest
Usage of Dataset
Used for predicting forest fire risk based on vegetation health and climatic conditions.

Dataset Information
Dataset Name:
Nallamala Forest Fire Prediction Dataset
Source:
MODIS Satellite (NDVI) and NASA POWER Weather Data
Domain:
Environmental Monitoring / Remote Sensing / Machine Learning
Task:
Forest Fire Prediction
Problem Type:
Binary Classification (Fire / Non-Fire)
File Format:
CSV
Dataset Link:
MODIS (NASA) and NASA POWER official portals
https://drive.google.com/drive/folders/1SLwdLpQZJAQgsIbkG5oeggeErkMsapX8


Dataset Overview
Total Records:
Multi-year time-series records (2012–2025)
Labeled Records:
All records labeled using satellite-detected fire pixels
Classes:
Fire, Non-Fire
Annotation Type:
Satellite-based labeling (MODIS & VIIRS fire pixels)
Why This Dataset?
• Captures vegetation health using NDVI
• Includes long-term climatic patterns
• Suitable for early forest fire risk prediction

Features Used
Feature 1: NDVI (Vegetation Index)
Feature 2: Temperature
Feature 3: Relative Humidity
(Additional: Solar Radiation, Lag Features)

Summary
This dataset integrates satellite vegetation indices and weather parameters to enable accurate and interpretable early forest fire prediction for the Nallamala Forest region.
