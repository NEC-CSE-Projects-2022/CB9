About the Project

NallaFireNet is an interpretable machine learning framework for early forest fire prediction in India’s Nallamala Forest. It integrates remote sensing NDVI data with synthetic weather variables from NASA POWER to forecast fire risk with high accuracy. By combining LightGBM, XGBoost, and Random Forest in a stacked ensemble, the system provides early alerts to support proactive forest management and ecological safety.

Core Methodology

  1. Data acquisition from MODIS NDVI and NASA POWER synthetic weather feeds (2012–2025).
  2. Feature engineering using temporal lags for temperature and humidity.
  3. Preprocessing with linear interpolation, normalization, and correlation analysis.
  4. SMOTE balancing to handle rare fire-day instances and improve recall.
  5. Stacked ensemble modeling using LightGBM, XGBoost, and Random Forest.
  6. Evaluation via Accuracy, Precision, Recall, F1-score, ROC, and visual analytics.

Key Advantages

  1. Achieves 91.8% accuracy and 69% recall for rare fire events.
  2. Combines multi-source satellite and weather data for predictive strength.
  3. Enhances interpretability via feature importance and NDVI trend analysis.
  4. Provides early warning capability for proactive fire prevention.
  5. Scalable to other wildfire-prone regions across India.
