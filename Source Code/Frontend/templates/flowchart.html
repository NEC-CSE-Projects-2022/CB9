Proposed System – NallaFireNet

The NallaFireNet architecture is a modular, interpretable machine learning framework for forest fire prediction in the Nallamala Forest. It integrates satellite-based NDVI imagery with synthetic meteorological data (temperature, humidity, and solar radiation) to forecast fire occurrences in advance. The pipeline includes data preprocessing, lag-based feature engineering, SMOTE balancing, and a stacked ensemble of LightGBM, XGBoost, and Random Forest models. The final model outputs reliable early-warning signals for proactive forest management and wildfire mitigation.
